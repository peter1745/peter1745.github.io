The Mono Guide

As some of you might know this website used to host a guide that I was writing that was meant to help people get started with the Mono library.
A few months ago I decided to abandon the guide, mainly because I lost interest in developing it further. I would like to change the guide into a video format in the future, but I currently don't know when I would have time to sit down and record a bunch of videos.

So you can consider this the official notice that the Mono guide will no longer be actively developed on this website. I've still ensured that it's available, you can visit this link if you're interested in reading it, or you can find it under the "Projects" tab.
